# <a href="/search?keynode=FreeCAD%20Main%20Window" target="_blank">FreeCAD Main Window</a> Sections

- <a href="https://wiki.freecad.org/images/5/5c/FreeCAD_022_Interface.png" target="_blank">View UI illustration</a>

1. **Main view area**
   - Contains tabbed windows

2. **3D view**
   - Central element for object display & manipulation
   - Multiple views/documents possible
   - Views can be detached

3. **Model and Tasks tab**
   - **Model tab** → shows document contents & structure
   - **Tasks tab** → prompts for tool/workbench-specific values

4. **Property editor**
   - Appears with Model tab active
   - **Data section** → parametric properties
   - **View section** → visualization properties

5. **Selection view**
   - Shows selected objects or sub-elements (vertices, edges, faces)

6. **Report view**
   - Displays messages, warnings, and errors

7. **Python console**
   - Prints executed commands
   - Allows entering Python code

8. **Status bar**
   - Shows messages and tooltips

9. **Toolbar area**
   - Docked toolbars

10. **Workbench selector**
    - Choose active workbench

11. **Standard menu**
    - Basic program operations
