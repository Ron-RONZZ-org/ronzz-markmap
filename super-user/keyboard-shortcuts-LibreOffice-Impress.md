# LibreOffice Impress — Useful Keyboard Shortcuts

- Navigation & workflow
  - next placeholder : `Ctrl + Enter`
  - enter group : `F3`
  - exit group : `Ctrl + F3`
  - duplicate object : `Shift + F3`
  - navigator : `Ctrl + Shift + F5`
  - spelling : `F7`
  - thesaurus : `Ctrl + F7`

- Editing & formatting
  - edit text : `F2`
  - edit points (vector editing) : `F8`
  - fit text to frame : `Ctrl + Shift + F8`
  - position & size dialog : `F4`

- Slide show control
  - start slideshow : `F5`
  - end slideshow : `Esc`
  - next effect / next slide :
    - `Left click`
    - `Space`
    - `Right Arrow`
    - `Down Arrow`
    - `Page Down`
    - `Enter`

- Text editing (common shortcuts)
  - bold : `Ctrl + B`
  - italic : `Ctrl + I`
  - underline : `Ctrl + U`
  - undo : `Ctrl + Z`
  - redo : `Ctrl + Y`
  - copy : `Ctrl + C`
  - paste : `Ctrl + V`
  - select all : `Ctrl + A`

- Slide management
  - new slide : `Ctrl + M`
  - duplicate slide : select slide, `Ctrl + C Ctrl + V`
  - delete slide : `Ctrl + Shift + Del`
  - move slide up/down : `Ctrl + ↑` / `Ctrl + ↓`

- View modes
  - normal view : `Ctrl + F1`
  - slide sorter : `Ctrl + F2`
  - notes view : `Ctrl + F3` (context‑dependent)

- Drawing & objects (shared with Draw)
  - select tool : `Esc`
  - constrain movement (horizontal/vertical) : `Shift` (while dragging)
  - duplicate while dragging : `Ctrl` (drag‑copy)
